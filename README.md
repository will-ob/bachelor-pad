
Batchelor Pad
================


Setup
---------------
```
$ ./unpack
```
Files
---------------
* `.bashrc` all the bashes
* `.bash_profile` just the login bashes
* `.vim` *the* editor
* `.tmux.conf` break-away terminal windows
* `.aliases` cause `bundle exec forman run rails console` is too long to type
* `.dotjs` to take over the internet
* `.bin` handy scripts
